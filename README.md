 Task 4: Sentiment Analysis on Social Media Data

In this Task 4 of the Data Science Internship at Prodigy InfoTech, I performed Sentiment Analysis on social media text data to classify user opinions as Positive, Negative, or Neutral. The project demonstrates how Natural Language Processing (NLP) techniques can be used to analyze public sentiment from textual data.

Tools Used: Python, Pandas, TextBlob, Matplotlib, and Google Colab

 Dataset: A sample social media dataset containing text posts was created and used to perform sentiment analysis.

 Steps Performed:

* Imported the required Python libraries.
* Created and loaded the social media text dataset.
* Performed sentiment analysis using the TextBlob library.
* Classified each post as Positive, Negative, or Neutral.
* Counted the number of posts in each sentiment category.
* Visualized the sentiment distribution using Bar Chart and Pie Chart.
* Analyzed the overall sentiment patterns from the dataset.

Output: Successfully classified social media posts into Positive, Negative, and Neutral sentiments and visualized the overall sentiment distribution through graphical representations.
